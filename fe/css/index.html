What is CSS?

CSS or Cascading Style Sheets adds styling such as fonts, colours or layouts to your website making the website more aesthetically pleasing. CSS uses a simplistic structure as well, making the learning curve for mastery relatively easy.

In short, HTML allows you to build the structure of your website and CSS makes that structure come to life.
