HTML

What is HTML?

HTML (Hyper Text Markup Language) is a web-based language that tells browsers how to showcase text and images on a webpage. It uses tags to indicate the various elements, determine the placement of paragraphs, headings, add images and videos.

As a Generalist

HTML and CSS are the frameworks used to build websites. Despite how complex websites and apps may be, they are still built with the fundamental blocks of the web: HTML & CSS.
I've learned enough to build this entire web page in HTML and CSS.

For example, our team's attempt to build a better user interface for Recoilize was a great experience to use whatever we've learned into practice. Recoilize is a Chrome Dev Tool used for debugging applications built with the new Recoil, an experimental library for React introduced by Facebook engineer Dave McCabe to meet state management needs that were lacking by Redux.
